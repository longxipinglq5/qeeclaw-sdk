# Contributing

感谢你关注 QeeClaw SDK。

## 贡献方式

欢迎以下类型的贡献：

- Bug 修复
- 文档改进
- 示例补充
- 新平台或新语言的接入样例
- 测试与验证脚本增强

## 提交建议

提交前建议先完成：

1. 阅读 `README.md` 与 `docs/`
2. 确认改动属于正确模块
3. 尽量补充示例、说明或测试
4. 避免把真实生产地址、密钥、个人路径写入公开仓库

## 开发约定

- `core-sdk` 以稳定 API 为优先
- `product-sdk` 可以继续演进，但要保持文档同步
- `runtime-sidecar` 属于实验性本地运行时，改动时请特别关注兼容性
- `meeting_device_firmware` 属于社区预览样例，改动时请优先保证公开示例可理解

## Pull Request 建议

- 标题尽量简洁明确
- 描述中说明改动动机、影响范围和验证方式
- 如果改动了接口、环境变量或公开文档，请同步更新相关文档

## 不建议提交的内容

- 真实生产域名作为默认值
- 真实 token、密钥、证书
- 与个人机器强绑定的绝对路径
- 未说明用途的大体积二进制文件
